# 🐄 小牛解谜 - 逻辑挑战

一款类似数独的颜色逻辑解谜游戏，共50个关卡等你挑战！

## ✨ 游戏特色

- 🎯 **10×10 大棋盘** - 更复杂的逻辑挑战
- 📊 **50个关卡** - 难度递增，从简单到烧脑
- 🧠 **数独式逻辑** - 类似数独的规则，但更有趣
- 🎨 **颜色区域** - 每种颜色区域都要放置小牛
- 🐄 **可爱小牛** - 萌萌的小牛作为游戏主角

## 📜 游戏规则

### 1. 每种颜色 1 头小牛
每个颜色区域都必须恰好放置 **1** 头小牛，不能多也不能少。

### 2. 行列唯一
- 任意一行里，只能出现 **1** 头小牛
- 任意一列里，也只能出现 **1** 头小牛
- （和数独中"每行每列数字不重复"的逻辑一致）

### 3. 小牛不能相邻
- 任意两头小牛不能在**上下左右**方向上相邻
- **斜对角**相邻是允许的 ✅

## 🎮 游戏玩法

1. 点击棋盘上的任意格子放置小牛 🐄
2. 如果位置合法，小牛会成功放置
3. 如果位置不合法（违反规则），格子会抖动提示
4. 填满所有颜色区域后，点击"检查答案"验证
5. 通关后进入下一关！

**辅助功能：**
- ✅ **检查答案** - 验证当前布局是否正确
- 🗑️ **清空棋盘** - 重新开始当前关卡
- 💡 **提示** - 高亮显示一个正确答案的位置
- ⏭️ **下一关** - 跳过当前关卡

## 📝 算法说明

### 关卡生成算法
1. 使用BFS（广度优先搜索）生成颜色区域
2. 每个区域大小均匀分配，形状随机
3. 确保所有格子都被分配到某个颜色区域

### 求解算法
使用回溯算法（Backtracking）生成有效解：
```javascript
function solve(regionIdx) {
    if (regionIdx >= regions.length) return true;
    
    const region = regions[regionIdx];
    for (const cell of region.cells) {
        if (canPlace(cell.row, cell.col)) {
            placeCow(cell.row, cell.col);
            if (solve(regionIdx + 1)) return true;
            removeCow(cell.row, cell.col);
        }
    }
    return false;
}
```

### 验证逻辑
- 行检查：确保每行只有1头牛
- 列检查：确保每列只有1头牛
- 相邻检查：确保上下左右没有相邻的牛
- 颜色检查：确保每个颜色区域恰好1头牛
